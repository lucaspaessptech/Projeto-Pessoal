Álgebra é o ramo da matemática que estuda a manipulação formal de equações, operações matemáticas, polinômios e estruturas algébricas. A álgebra é onde você realmente começa a desenvolver raciocínio e reconhecer padrões, pois é nessa parte que você tem que se esforçar para encontrar as manipulações necessárias e correta para resolver uma equação
Inclui tópicos como:

Propriedades das operações matemáticas; Expressões algébricas;
Equação primeiro grau;
Equação segundo grau;
Funções;
Gráficos das funções;
